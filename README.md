# Butterflies Site

A Next.js storefront-style landing page for Butterflies apparel and slides.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Project Structure

- `src/app/` - App Router pages, layout, and global styles.
- `src/components/` - Reusable page sections and product components.
- `src/data/` - Product content used by the storefront.
- `public/images/` - Website image assets.

## Deploy

This app can be deployed on Vercel or any host that supports Next.js.
